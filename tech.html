The core of this site is built using semantic HTML5, modern CSS3 with custom variables for theme switching, and Vanilla JavaScript for logical functionality like cursor followers and scroll observers.

The custom cursor follower and scroll-triggered animations (typing effect, fade-ins) are implemented using Vanilla JavaScript and native APIs like IntersectionObserver. This minimizes external dependencies, boosting both maintainability and performance.

For complex visual effects, the site utilizes Three.js (WebGL) to run high-performance shaders directly on the GPU. The particle system in 'Fluid Cosmos' uses the native Canvas 2D API combined with a spatial grid optimization algorithm.

The 'Kaleidoscope of Sound' experience leverages the native Web Audio API to analyze microphone input in real-time, mapping audio frequencies to visual parameters.
